# 오늘의 핑계 생성기

## 1. 서비스 개요
이 서비스는 사용자가 상황을 선택하면
회사에서 실제로 사용할 수 있을 법한
그럴듯한 핑계 문장을 하나 생성해주는 웹페이지이다.

웃기지만 너무 가볍지 않고,
변명처럼 보이지만 최소한의 책임감은 느껴지는 톤을 목표로 한다.

---

## 2. 대상 사용자
- 직장인
- 개발자가 아닌 일반 사용자
- 모바일 / PC 모두 사용 가능해야 함

---

## 3. 핵심 기능

### 3.1 상황 선택
사용자는 아래 상황 중 하나를 선택할 수 있다.
- 지각
- 업무 지연
- 연락 늦음
- 실수함

### 3.2 핑계 생성
- "핑계 생성하기" 버튼을 누르면
- 선택한 상황에 맞는 핑계 문장이 랜덤으로 1개 출력된다.
- 핑계 문장과 어울리는 캐릭터 사진 1개가 같이 출력된다

### 3.3 핑계 톤 선택 (선택 기능)
사용자는 핑계의 톤을 선택할 수 있다.
- 😇 무해한 핑계
- 😐 중립적인 핑계
- 😈 뻔뻔한 핑계
- 😎 최신 유행어 핑계

---

## 4. UI / UX 요구사항
- 다크 모드 기반
- 중앙 정렬된 카드 UI
- 미니멀하고 깔끔한 디자인
- 살짝 위트 있는 문구 사용
- 모바일 화면에서도 깨지지 않을 것

---

## 5. 기술 요구사항
- HTML / CSS / JavaScript 파일 분리 가능
- 프레임워크 사용 여부는 구현자가 판단
- 과도한 설정이나 복잡한 빌드 과정은 지양
- 코드 가독성 중요
- 유지보수 가능한 구조

---

## 6. 기타
- 로그인 기능 없음
- 결과 데이터 링크 공유 기능
- 결과는 새로고침 시 초기화되어도 무방
